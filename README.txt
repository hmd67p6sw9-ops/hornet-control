Hornet Control v1.3.3 — автоматичний ID борта

Що змінено:
- користувач більше не вводить ID борта;
- система знаходить найбільший зайнятий номер;
- новому борту призначається наступний номер;
- поле ID у формі доступне лише для перегляду;
- остаточний ID повторно визначається на сервері під блокуванням;
- два одночасні запити не отримають однаковий ID;
- захист від однакових серійних номерів збережено;
- після створення автоматично відкривається QR нового борта.

Приклад:
- останній найбільший ID HN-0018;
- система запропонує HN-0019;
- якщо HN-0099 існує, наступним буде HN-0100.

Встановлення:
1. У Code.gs встав чинний API_KEY.
2. Повністю заміни Code.gs в Apps Script.
3. Збережи та опублікуй нову версію.
4. У index.html встав чинні API_URL та API_KEY.
5. Повністю заміни index.html у GitHub.
6. Повністю заміни sw.js.
7. Commit changes.
8. Закрий PWA на iPhone та відкрий заново.
